# Customer Behavior Analysis

## Overview

This project analyzes customer shopping behavior using transactional data to uncover insights about purchasing patterns, revenue trends, and product performance.
The project follows an **end-to-end data analytics workflow** including data cleaning in Python, SQL-based analysis, and visualization through a Power BI dashboard.

## Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* PostgreSQL / SQL
* Power BI
* Gamma (Presentation)

## Project Workflow

1. **Data Cleaning & EDA (Python)**

   * Loaded and explored the dataset using Pandas
   * Handled missing values and duplicates
   * Performed exploratory data analysis to understand trends

2. **SQL Data Analysis**

   * Loaded cleaned data into PostgreSQL
   * Wrote SQL queries to analyze customer purchasing behavior and revenue trends

3. **Power BI Dashboard**

   * Built an interactive dashboard to visualize sales performance and customer insights
