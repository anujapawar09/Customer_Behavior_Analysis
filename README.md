# Customer_bheavior_analysis

## Overview
This project demonstrates a complete **data analytics workflow**, starting from raw data processing to visualization and reporting. The project includes loading a dataset using **Python**, performing **Exploratory Data Analysis (EDA)**, cleaning and transforming data, executing **SQL queries** across multiple database systems, and building an interactive dashboard in **Power BI**.

The final insights were documented in a detailed report and presented through a **PowerPoint presentation created using Gamma**.

---

## Dataset
The dataset used in this project contains **transactional data** including customer details, purchase behavior, product information, and order-related attributes.

### Key Features of the Dataset
- **Customer ID**
- **Product Category**
- **Purchase Amount**
- **Shipping Type**
- **Order Date**
- **Payment Method**
- **Customer Location**

This dataset was used to analyze **customer purchasing patterns, revenue trends, and operational insights**.
